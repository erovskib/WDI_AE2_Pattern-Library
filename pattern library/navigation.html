Navigation

The navigation component will be used to navigate the pattern library.

The navigation is completly responsive and it breaks down after the browser width becomes less than 750px this can be changed in the css media query. *Resize the browser to see the reponsivness*

HTML

<nav id="navtop">

        <label for="touch"><span>Menu</span></label>
        <input type="checkbox" id="touch">

        <ul class="slide">
          <li><a href="navigation.html">Navigation</a></li>
          <li><a href="header.html">Header</a></li>
          <li><a href="buttons.html">Button Types</a></li>
          <li><a href="jumbotron.html">Jumbotron</a></li>
          <li><a href="footer.html">Footer</a></li>
          </li>
          <li><a href="other.html">Other</a></li>
          </li>
        </ul>
</nav>
    </div>

              * {

body {

/* Styles Desktop */

#navtop label,
#navtop input {
  display: none;
}

.slide li {
  /* Styles liens desktop */

  height: 70px;

.slide li a {

/* Styles Mobile - > 750px - */

@media (max-width: 750px) {
  #navtop label,
  #navtop input {
    display: block;
  }

  #navtop span {
    height: 70px;
    padding: 25px;
    color: #eaeaea;
    font-size: 1.2em;
    display: block;
  }

  #navtop span::after {
    float: right;
    right: 10%;
    content: "+";
  }

  .slide {
    clear: both;
    background: #d9d9d9;
    width: 100%;
    height: 0px;
    overflow: hidden;
    text-align: center;
    transition: height 0.3s ease;
  }

  .slide li {
    /* Styles liens mobile */
    display: block;
    width: 100%;
  }

  .slide li a {
    color: #000000;
  } /* Coulour liens mobile */

  #touch {
    position: absolute;
    opacity: 0;
    height: 0px;
  } /* Checkbox invisble */

  #touch:checked + .slide {
    height: 420px;
  }
